📰 Retro Forum

🔗 Live Site: https://retro-forum-ruby.vercel.app

A modern retro-styled forum web application that allows users to explore posts, view discussions, and interact with a clean, responsive UI. Built with a focus on layout design, dynamic content rendering, and user-friendly browsing experience.

🚀 Features
🗂️ Forum-style post layout
🔎 Search and filter functionality
💬 Interactive post listing UI
📌 Dynamic content rendering
📱 Fully responsive design
🎨 Retro-themed modern UI styling
⚡ Smooth and lightweight frontend experience
🛠️ Technologies Used

HTML5
CSS3 / Tailwind CSS
JavaScript (Vanilla JS)

📁 Project Structure
├── images/               # Image assets
├── js/                   # JavaScript files
├── style.css             # Custom styles
├── index.html            # Main HTML file
├── tailwind.config.js    # Tailwind configuration
├── README.md             # Project documentation
└── .gitattributes

⚙️ Setup & Usage
Clone the repository:
git clone https://github.com/istewakhassantewak/retro-forum.git
Navigate to the project folder:
cd retro-forum
Open index.html in your browser.
